# 开发期发票样本

本目录存放真实发票 PDF，仅供本机开发与调试使用。

**重要约束（DESIGN.md §9）**：

- 真实发票含税号 / 抬头 / 姓名 / 身份证号等敏感数据。
- 本目录通过 `.gitignore` 整体排除入库，**不要提交**。
- 不要把内容拷贝到 issue / 截图 / 日志。
- 如要在测试中使用，请先脱敏后放入 `tests/fixtures/`。

当前样本：
- `普票1.pdf` / `普票2.pdf` — 数电普票
- `专票1.pdf` / `专票2.pdf` / `专票3.pdf` — 数电专票
- `12306-1.pdf` / `12306-2pdf.pdf` — 12306 电子客票
